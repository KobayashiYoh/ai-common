プルリクエストのコードレビューを実施し、レビューコメントを投稿します。

## 実行方法

```bash
ai/scripts/github/review-pr.sh [PR番号またはURL]
```

詳細は @ai/scripts/github/README.md を参照してください。

## レビュー観点

@docs/coding-rules/CODING_RULE_COMMON.md に従い：

1. **コーディング規約** - 命名規則、インデント、型安全性
2. **ロジック・設計** - 正確性、パフォーマンス、エッジケース
3. **テスト** - コードの有無とカバレッジ
4. **セキュリティ** - 脆弱性、認証・認可
5. **ドキュメント** - コメント、README 更新

## 基本方針

- 問題がない場合は無理に指摘しない
- 差分中心にレビュー
- 具体的で建設的な指摘
